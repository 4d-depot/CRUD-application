### A Practical Showcase of User Login and Data Management in Qodly

The **Qodly CRUD & Authentication Demo** is a lightweight but complete web application built to illustrate the fundamentals of **authentication** and **CRUD operations** using the **Qodly** low-code platform.

It serves as a clean learning resource for developers discovering Qodly or anyone needing a starting template for secure data-driven apps.

---

# üéØ Purpose of the Application

Modern applications rely on two essential elements:

- **Authentication** to secure user access  
- **CRUD operations** to manage data  

This demo app brings both concepts together into a simple, understandable structure.  
The goal of the application is to provide:

- A clear example of **login** flow  
- A protected dashboard accessible only after authentication  
- A complete CRUD interface (Create, Read, Update, Delete)  
- A basic data display (datable/matrix) 

The application is ideal for:

- Developers new to **Qodly**  
- Teams building internal tools  
- Users learning authentication and data management  
- Anyone needing a clean boilerplate to start from

# üöÄ How to Integrate and Use the Application

Follow these steps to get the CRUD & Authentication Demo running inside your own Qodly environment.

---

## 1. Clone or Import the Template

Download or clone the project into your Qodly workspace from the provided GitHub repository.

---

## 2. Launch the Application

To access and start using the application, open the project in Qodly Studio and run the application.

You can choose your preferred starting page depending on your workflow:

Set the starting page to HomePage if you want an overview of the entire application and its features.
Set the starting page to home if you want to land directly on the login process.

Explore the different pages to discover all the available features of the system..

---

## 3. Customize as Needed

This demo is meant to be extended.  
You can modify or enhance:

- The **data model** (add new tables or fields)  
- The **authentication logic**  
- **UI components** with QodlyPages  
- **Additional CRUD modules**  
- Custom **security rules**  

Use this demo as your foundation for building full-scale applications in Qodly.
